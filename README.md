# code-challenge-99techteam
This repository contains solutions for the 99TechTeam code challenge:

- Problem 1: Algorithm / utility implementation with unit tests  
- Problem 2: Currency Swap web application built with Vite + React + TypeScript

Each problem is isolated in its own folder and can be run independently.
---

## Prerequisites

- Node.js >= 18
- npm
- Yarn (v3+ recommended)


Problem 1 – Run Tests
```bash
cd src/problem1
npm install
npm run test
```

Problem 2 – Run Application
```bash
cd src/problem2/currency-swap-app
yarn install
yarn dev
```
